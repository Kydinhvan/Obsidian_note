A well-defined machine learning task: < T, P, E>
- experience E, performance measure P, tasks T

## Example: 
### customer segments:
- T: Dividing a customer base into groups of individuals of similar trait
- P: Accuracy of the grouping  
- E: Set of customer profile data (e.g., demographics and purchasing behaviors) 
To describe intrinsic structure/property in data 

### Handwritten recognition 
- T:  Recognizing hand-written words  
- P:  Percentage of words correctly classified  
- E:  Database of images of handwritten words with human labels
To make predictions

### Credit card fraud detection
- T:  Determine whether a credit card transaction is fraud  
- P:  Percentage of false positives, percentage of false negatives  
- E:  Database of historic transactions (with labels)
To make predictions

### Playing games
- T: Having sequential control of the actions made  
- P: Game scores or W/L
- E: Sequential observations of game state  
To make sequential decisions
