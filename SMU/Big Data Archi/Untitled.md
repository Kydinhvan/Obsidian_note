A **ledger-based system** is a record-keeping structure that tracks the movement of value or data between accounts

## Batch processing vs Stream processing
- Batch processing handles large data chunks periodically for high throughput 
	- e.g., monthly reports
- Stream processing analyzes continuous data streams in real-time for low latency 
	- e.g., fraud detection
	- AWS kinesis data analytic
